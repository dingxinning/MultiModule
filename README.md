    - String 字符串各种操作

    - 通信 httpClient、RestTemplate、netty
    - 并发编程 concurrent
    - RX系列 RxJava...
    - 各种常用utils
    - ...

    - 参考 Spring In Action
5. 学习 Spring MVC

    - Employee (impl & 查询数据)

    - learn (MVC各种基本注解尝试)
    - ...
